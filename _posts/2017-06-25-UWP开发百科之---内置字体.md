UWP可以很方便的内置第三方字体。。。    
     
# 语法

语法嘛。。很简单   
*Font File Path* # *Font Name*    
    
# 例子

```
FontFamily="/Assets/SEGMDL2.TTF#Segoe MDL2 Assets"
```

这样就使用了当前项目目录下面的Assets文件夹里面的SEGMDL2.TTF字体文件并且索引到**Segoe MDL2 Assets**字体上
